Installation:

Step 1: 

 	unzip datagrid_nav20 and copy files to your apache htdocs folder.

Step 2: 

 	Import file "datagrid.sql" to your mysql server.

	Example:

	UNIX

	shell> mysql datagrid < datagrid.sql

	If you are using windows use this at command prompt:

	mysql -p -u[root] [datagrid] < datagrid.sql

Step 3:

	write "http://localhost/index.php?page=1&bindex=1&i=1" in your browser.

And your done!
